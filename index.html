Overview of Methodology

Learn how to configure settings for Scribbler, such as your syntax highlighting preference and the default saving folder location.

Installation

              
                $ scribbler  ——config
                  {
                    “encryption”: true, 
                    “highlighting“: true,
                    “prettyTable”: false,
                    “font”: [“Helvetica”, “sans-serif”],
                    “folder”: “~/Desktop“
                  }
